idea操作数据库
发表于 2020-03-21| 更新于 2020-03-21
阅读量:

说明

jetbrain全家桶中大部分是可以连接并且直接操作数据库的,比如idea、clion、pycharm、phpstorm,操作对初学sql语句的人来说十分友善

环境介绍

  • linux / windows
  • mysql服务
  • idea(或者其他)

打开mysql服务

这里以archlinux和mariaDb(mariadb和mysql基本通用)为例

打开idea

看到屏幕右边的database按钮,点开


点击" + ", 然后 datasource->mysql
填写数据库相关信息, host 填写 'localhost' 即可 , 或者也可以填写对应ip连接远程数据库
TestConnection 可能会出现这个 , 点击download 然后等待几秒即可
TestConnection如下即连接成功
然后就可以用idea操作数据库了,自动补全,而且代码都不会消失,配合phpadmin使用体验良好
