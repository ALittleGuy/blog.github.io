wordpress服务器搭建
发表于 2020-03-14| 更新于 2020-03-15|前端
阅读量:

说明

最近国内上github有点困难,速度比较慢,女生节时用github pages给女神们写的网页也有打不开的尴尬情况,而我的博客也是建立在github pages上的,在国内也不好打开,因此本期介绍如何用WordPress快速的搭建简单的网站


环境准备

  • 电脑一台即可

购买服务器

首先,搭建网站必须一台服务器,这里我选择阿里云的轻量应用服务器
网址:https://www.aliyun.com/product/swas?spm=5176.13910061.h2v3icoap.16.39cd6f69jOx6lH&aly_as=YuxJKuqc


服务器选择

地区建议选择香港,不用备案,应用镜像直接选择已有的WordPress,省下安装的功夫.



系统镜像则选用任意linux即可,windows需要更高的配置

选择最低配即可,后期可以加.
价格为每月24￥,一般学生党也可以承担


服务器配置

概览

打开控制台,发现Apache、mysql、php和WordPress 都已经安装了，但没有ftp
点击重置密码，重置系统的root密码,将密码记录下来


防火墙配置

防火墙各端口说明：

端口名称 说明 端口 要求
http 基本的http服务端口 默认80 必选
http 基本的https服务端口 默认443 必选
ssh ssh远程连接端口，ssh工具远程连接服务器 默认22 可选
MySQL mysql开放端口，允许远程连接数据库 默认3306 可选
FTP ftp服务端口（系统默认没有装ftp） 默认 21 可选
自定义 ftp浏览器服务需要 1024/65535 可选

WordPress密码获取

点击远程连接打开控制台，用root用户登录，复制WordPress信息中的那条命令，粘贴到控制台，获取密码


运行

网址

地址： http://your_ip (your_ip处填写你自己的服务器IP),用户名是admin ， 密码则为上面获取的那一个

设计网站

打开 http://your_ip/wp-admin (your_ip处填写你自己的服务器IP)


在这里就可以配置专属于你的网站,到这里基本的服务器就架好了,WordPress相对于hexo更加的强大,不仅仅是普通的博客,用途十分广泛.
