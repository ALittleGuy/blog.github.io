media creation tools 实现win10安装
发表于 2020-02-13| 更新于 2020-03-14|win10
阅读量:

说明

最近又又又又又有人找我刷系统,感觉我学的软件工程就是个修电脑的,但其实如今win10的安装是十分的简单的一件事,完全不需要什么PE系统balabala的,只需要一个U盘和一台用来制作启动盘的电脑,过程比较简单

准备工作

  • 一个8g以上的U盘
  • 一台装有windows的电脑

下载media creation tool

可参照这个链接
https://windows-media-creation-tool.en.softonic.com/

下载完毕后,打开软件即可

启动盘制作

选择windows 版本

选择制作启动盘

选择U盘

过程时间比较长,大概20分钟左右

进入bios

查询对应主办进入bios的按键,然后开机后进入bios,选择U盘启动

进入装机页面

选择一些相关的选项以及win10所在的分区

等待安装完成

天涯若比邻

配置后就得到一个崭新的win10
