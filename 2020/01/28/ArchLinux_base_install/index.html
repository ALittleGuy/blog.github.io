Arch Linux 的安装（虚拟机）

工欲善其身，必先利其器。好的操作系统可以极大的提高程序猿的工作效率，但是对于大部分学生来说——mac好贵！所以Linux，性价比约等于无限，是个十分实惠且高效的选择。

Arch linux 是2002年推出的发行版。linux有众多的发行版，为啥选择 Arch呢？理由如下：

  • 极高可定制性，兼容各种桌面
  • 极其强大说明文档 arch wiki
  • 支持滚动更新
  • 强大的社区软件库aur（arch user repository）

安装介绍

arch的优势十分突出，劣势也是，arch的安装和配置过程相对于刚开始使用linux的人来说比较复杂，不过这点劣势会随着用户的使用被逐渐磨平。
所以最好是先拿虚拟机练习一下，再到主机上安装，本篇博客也在虚拟机上进行工作。

系统环境

  • 虚拟机软件virtual box
  • 系统镜像archlinux
  • 20g左右的硬盘空间
