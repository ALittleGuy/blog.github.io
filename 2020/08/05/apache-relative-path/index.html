php相对路径的上级目录访问
发表于 2020-08-05| 更新于 2020-08-06|php
阅读量:

问题描述

如上图，一个YII框架项目的路径，其中 frontend 和 backend 两个端口各自有一个入口文件，但是资源文件存储在 img 文件夹中，在html中一般使用相对路径来调用资源文件，但是相对路径的上级目录如何访问呢。

解决方案

首先这种情况下使用绝对路径是不行的，假如 img/123.jpg的绝对路径为 /www/test.com/www/img/123.jpg，那么在<img>标签中的src填写绝对路径时返回的是http://localhost/www/test.com/www/img/123.jpg 这条路径显然不能显示正常的图片。

那么使用相对路径呢，用../img的方法的访问img文件夹，根据我的试验，在YII框架中直接使用 ../123.jpg 可以查看www/123.jpg这个文件，但是如果使用 ../img/123.jpg 并不凑效，即能访问上级目录中的文件但是无法访问上级目录中的文件夹

那么最终的解决方案是修改apache的配置，定义路径的别名，打开http.conf文件,定义一个路径的别名，如图

Alias /img "www/test.com/www/img"

这样，访问https://localhost/img 的时候访问的就是img文件夹,在<img>标签中填写’img/123.jpg’就可以访问img文件夹下123.jpg
